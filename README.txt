1. Commands
- /gg (custom GG color)
- /convert [.xxx] (converts the coordinates, for example /convert .123 will print out .877

2. Settings
- The BetterLC settings can be found by pressing ESC and then clicking on the "BetterLC" Button (top left)

3. Keys
- Keys can be changed in the controls menu under the BetterLinkCraft category. 
- Control Presets can be found in the Controls Menu.

4. Chat Macros
- Chat Macros in the BLC Settings in Chat Options -> AutoText
